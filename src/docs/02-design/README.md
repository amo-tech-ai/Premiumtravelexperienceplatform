## Design Principles

### Calm Luxury
- Editorial spacing (generous whitespace)
- Soft cards with subtle shadows
- Confident typography (clear hierarchy)
- Premium micro-interactions
- Trust-building (explainable AI)

### Mobile-First
- Touch-optimized targets (>44px)
- One-hand usable
- Bottom sheets for modals
- Floating action buttons
- Swipe gestures

### Accessibility
- WCAG AA compliance
- Keyboard navigation
- Screen reader support
- Focus states
- Color contrast
