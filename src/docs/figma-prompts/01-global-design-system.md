# Figma Prompt 0.1: Global Design System

**Context:**
You are setting up the foundational visual language for a "Luxury AI Concierge" application for Medellín. The aesthetic is **"Luxury Editorial" meets "Eco-Modern"**. It should feel sophisticated, trustworthy, and calm—not flashy or tech-heavy.

**Instructions:**
Create a Figma "Design System" frame defining the following global styles and variables.

## 1. Color Palette (Strict)
*   **Primary (Emerald):** `#064E3B` (Use for primary buttons, active states, branding).
*   **Primary Hover:** `#043D2E` (Darker shade).
*   **Accent (Gold):** `#D4AF37` (Use for icons, highlights, premium badges. Muted/Metallic, NOT warning yellow).
*   **Background (Canvas):** `#F7F7F5` (Off-white/Cream. Use for app background).
*   **Surface (Cards/Modals):** `#FFFFFF` (Pure white).
*   **Text Main:** `#111827` (Charcoal/Almost Black. Never pure black).
*   **Text Muted:** `#6B7280` (Cool Grey).
*   **Status (Verified):** `#059669` (Emerald Green).
*   **Status (Error):** `#991B1B` (Deep Red).

## 2. Typography
*   **Headings (Display):** `Playfair Display` (Serif).
    *   *Usage:* Headlines, Hero text, Section titles.
    *   *Weights:* Bold, Semibold.
*   **Body (UI):** `Inter` (Sans-serif).
    *   *Usage:* UI elements, paragraphs, buttons, inputs.
    *   *Weights:* Regular, Medium.

## 3. Spacing & Shape
*   **Grid:** 4px baseline grid.
*   **Border Radius:**
    *   `Small (4px)`: Buttons, Inputs (Refined look).
    *   `Medium (8px)`: Cards, Images.
    *   `Large (16px)`: Modals, Bottom Sheets.
    *   `Full (999px)`: Tags, Pills, Avatars.

## 4. Shadows & Effects
*   **Elevation:** Soft, diffused shadows. Avoid harsh black.
    *   *Subtle:* `0px 2px 4px rgba(6, 78, 59, 0.05)` (Emerald hint).
    *   *Floating:* `0px 10px 15px -3px rgba(0, 0, 0, 0.1)`.

## 5. Components to Design (Reference)
Create these basic components to test the system:
1.  **Primary Button:** Emerald background, White text, 4px radius.
2.  **Ghost Button:** Transparent, Emerald text, Gold hover effect.
3.  **Input Field:** White background, thin grey border, Inter text.
4.  **Tag/Chip:** Off-white background, Charcoal text, 1px Gold border (optional).

**Output Goal:**
A clean, organized style guide that allows for rapid UI assembly in future steps.
