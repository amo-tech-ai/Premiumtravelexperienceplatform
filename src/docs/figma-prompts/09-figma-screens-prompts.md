### Common Elements
- **Color Scheme:** Primary (#064E3B), Accent (#FBBF24)
- **Typography:** Playfair Display (headers), Inter (body)
- **Spacing:** 4px base unit
- **Radius:** 12px for cards
- **Shadow:** Soft (0 2px 8px rgba(0,0,0,0.08))

### Responsive Breakpoints
- Mobile: 375px - 767px
- Tablet: 768px - 1023px
- Desktop: 1024px+
