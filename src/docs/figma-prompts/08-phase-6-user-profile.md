# Figma Phase 6: User Profile & Settings

**Goal:** A personalized hub for the user to manage their identity, preferences, and history.
**Dependency:** Linked from the Global Footer or Home Header.

---

## Prompt 6.1: Profile Dashboard (The Hub)

**Context:**
The landing page when a user clicks "My Account". It provides a high-level overview and quick access to key sections.

**Instructions:**
Design the **Profile Dashboard**.

### 1. Header / Identity
*   **Layout:** Centered column.
*   **Avatar:** Large circle (`100px`) with an edit pencil icon overlay.
*   **Name:** Large Serif headline (e.g., "Welcome, Sofia").
*   **Member Status:** Small Gold badge: "Concierge Member" or "Local Explorer".

### 2. Quick Actions Grid
*   **Layout:** 2x2 or 3x1 Grid of cards.
*   **Cards:**
    *   **My Trips:** Icon + "3 Upcoming".
    *   **Favorites:** Icon + "12 Saved Places".
    *   **Wallet:** Icon + "Payment Methods".

### 3. "Favorites" Preview
*   **Section Title:** "Recently Saved".
*   **Content:** A horizontal scrolling list of small square cards (Restaurants/Events users liked).

---

## Prompt 6.2: Settings & Preferences

**Context:**
Where users configure the app mechanics.

**Instructions:**
Design the **Settings Screen**.

### 1. Layout
*   **Style:** Clean, divided list rows (iOS Settings style but cleaner).
*   **Row Architecture:**
    *   **Left:** Icon + Label (Medium weight Inter).
    *   **Right:** Value (Grey text), Toggle Switch, or Chevron arrow.

### 2. Sections
*   **Account:** Edit Profile, Password, Security.
*   **Preferences:**
    *   *Currency:* USD / COP selector.
    *   *Language:* English / Spanish.
    *   *Notifications:* Toggles for "Booking Updates", "Recommendations", "Marketing".
*   **Support:** Help Center, Chat with Human, Legal.
*   **Danger Zone:** Delete Account (Red text, centered at bottom).

---

## Prompt 6.3: My Trips (Itinerary Management)

**Context:**
A detailed view of what the user has booked.

**Instructions:**
Design the **Bookings List**.

### 1. Navigation Tabs
*   **Tabs:** "Upcoming" (Active) | "History" (Inactive).

### 2. Booking Cards (Status Driven)
*   **Layout:** Stacked list of wide cards.
*   **Visuals:**
    *   **Status Badge:** Top right.
        *   *Confirmed:* Emerald Green background, White text.
        *   *Pending:* Amber/Gold background.
        *   *Past:* Grey background.
    *   **Thumbnail:** Left side, square image.
    *   **Details:** Title, Date/Time, Location.
*   **Action Buttons (Bottom of card):**
    *   "View Ticket" (Primary).
    *   "Get Directions" (Secondary, Icon only).
    *   "Cancel" (Text link, muted).

### 3. Empty State
*   **Visual:** A line-art illustration of a suitcase or map.
*   **Text:** "No upcoming trips."
*   **CTA:** "Start Planning" (Links back to Home/Chat).
